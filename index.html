Google Maps API Key

You need a Google Maps JavaScript API key to load the satellite map of Honiara. The key is free for up to 28,000 map loads per month — well within your fieldwork needs.

1. Go to console.cloud.google.com
2. Create a new project (e.g. "Honiara Survey Portal")
3. Go to APIs & Services → Library and enable Maps JavaScript API
4. Go to APIs & Services → Credentials → Create API Key
5. Copy the key and paste it below

The key is stored in your browser session only and never sent anywhere except to Google Maps.
